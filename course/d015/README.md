## What I learned(Important)

* for statement
  * how to use:

  ```java
  for($inital-state$ ; $condition$ ; $state-change$) {
    $action$
  }
  ```
  
  * statement order:  
    1. $inital-state$
    2. $condition$
    3. if true: $action$ -> $state-change$, if false: **ended**
    4. if **not ended**: go to 2.

* while statement
  * how to use:

  ```java
  while($condition$) {
    $action code$
  }
  ```

* do-while statement
  * how to use:

  ```java
  do {
    $action code$
  } while($condition$); // you MUST ended with semicolon!
  ```
